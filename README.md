# MusicPlayerApp

## Instalação

Clone o projeto e acesse a sua pasta:

```sh
$ git clone https://github.com/thiagoemanoel98/MusicPlayerApp.git
$ cd MusicPlayerApp
```

Instale as dependências do projeto e inicie:

```sh
# Dependências
$ npm install
   ou
$ yarn

# Start Android
$ npx react-native run-android
   ou
$ yarn android

# Start IOS
npx react-native run-ios
   ou
$ yarn android
```
